# 💸 Карманный бот-бухгалтер (и конвертер валют)

Надоело записывать свои траты на кофе и подписки в стандартные заметки, поэтому накидал простенького бота для телеги. Заодно прикрутил конвертер валют, потому что постоянно приходится переводить доллары/евро в рубли при покупках в интернете.

Написано на Python за пару вечеров. Вся логика специально разбита ровно на 4 файла, чтобы код не превратился в сплошную лапшу и в нем можно было легко разобраться.

## Что умеет
* **Записывать расходы** (сумма + категория).
* **Считать общую сумму** потраченного (пока что за всё время, фильтры по месяцам добавлю позже, если дойдут руки).
* **Конвертировать валюты** по актуальному курсу (используется открытая апишка er-api).

## Как это выглядит внутри
Специально не стал городить сложную архитектуру. Тут всего 4 файла:
* `main.py` — запуск бота и хэндлеры команд.
* `logic.py` — тут живет конвертер валют и запросы к API.
* `database.py` — работа с SQLite (сохраняем траты).
* `config.py` — настройки и токен.

## Как запустить у себя

1. Скачиваем или клонируем репу.
2. Ставим нужные либы. У вас должен быть установлен Python 3.8+.
   ```bash
   pip install pyTelegramBotAPI requests
   ```
3. Идем к[@BotFather](https://t.me/BotFather) в Телеграме, создаем нового бота и копируем его токен.
4. Открываем файл `config.py` и вставляем токен вместо `YOUR_BOT_TOKEN`.
5. Запускаем:
   ```bash
   python main.py
   ```
   *При первом запуске бот сам создаст файлик `expenses.db` (база данных) рядом со скриптом, так что ничего дополнительно настраивать не надо.*

## Команды бота
Кидайте прямо в чат с ботом:

* `/start` или `/help` — выведет шпаргалку.
* `/add 500 шаурма` — добавит расход. Категорию можно писать любую, хоть одним словом, хоть с эмодзи.
* `/stats` — покажет, сколько всего денег вы уже спустили.
* `/convert 100 USD RUB` — переведет 100 баксов в рубли (тикеры валют можно писать и маленькими буквами, бот поймет).
